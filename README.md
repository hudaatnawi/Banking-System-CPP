# 🏦 Banking System in C++

A simple console-based Banking Management System built using C++.  
This project demonstrates file handling, structured programming, and menu-driven user interaction.

---

## 📌 Features

- Add new clients
- Delete existing clients
- Update client information
- Search for a client
- Display all clients
- Deposit money
- Withdraw money
- View total balances
